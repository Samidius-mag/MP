Миграции БД
============

Ручной запуск миграций:

1) Настройте переменные окружения в `server/.env` (DB_HOST, DB_PORT, DB_NAME, DB_USER, DB_PASSWORD).
2) Выполните:

```bash
npm run migrate
```

Скрипт выполнит все `.sql` файлы из директории `server/migrations` в алфавитном порядке в одной транзакции на файл.
