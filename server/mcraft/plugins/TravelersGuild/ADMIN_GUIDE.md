# Руководство администратора TravelersGuild

## Выдача прав администратора

### Способ 1: Через консоль сервера

Выполните в консоли сервера:
```
op <ник_игрока>
```

Например:
```
op Samidius
```

Это даст игроку статус оператора (OP), который автоматически получает все права, включая `travelersguild.admin`.

### Способ 2: Через команду в игре (если вы уже OP)

Если вы уже оператор, выполните:
```
/op <ник_игрока>
```

### Способ 3: Через файл ops.json

1. Остановите сервер
2. Откройте файл `server/mcraft/ops.json`
3. Добавьте запись:
```json
[
  {
    "uuid": "uuid-игрока",
    "name": "Ник_игрока",
    "level": 4,
    "bypassesPlayerLimit": false
  }
]
```
4. Запустите сервер

### Способ 4: Через плагин управления правами (если установлен)

Если у вас установлен плагин управления правами (например, LuckPerms):

```
/lp user <ник_игрока> permission set travelersguild.admin true
```

## Проверка прав

Чтобы проверить, есть ли у игрока права администратора:

```
/op list
```

Или в игре:
```
/plugins
```

## Команды администратора

После получения прав администратора доступны команды:

- `/guildadmin spawn` - Создать NPC-заведующего в текущей локации
- `/guildadmin reload` - Перезагрузить данные гильдии

## Устранение проблем

### Команда /guildadmin не работает

1. Проверьте, что плагин загружен:
   ```
   /plugins
   ```

2. Проверьте права:
   ```
   /op list
   ```

3. Перезагрузите плагин:
   ```
   /reload confirm
   ```
   Или перезапустите сервер

4. Проверьте логи:
   ```
   tail -f logs/latest.log
   ```

### NPC не создается

1. Убедитесь, что вы находитесь в правильном мире
2. Проверьте, что у вас есть права на создание сущностей
3. Попробуйте выполнить команду еще раз

## Сброс прав

Чтобы убрать права оператора:

```
/deop <ник_игрока>
```

Или удалите запись из `ops.json` и перезапустите сервер.
