# Datapack: Отображение времени игрового мира

Этот datapack автоматически отображает текущее время игрового мира в правом углу экрана через scoreboard.

## Установка

Datapack уже установлен в папке `server/mcraft/world/datapacks/gametime_display/`.

При запуске сервера datapack автоматически загрузится и начнет работать.

## Как это работает

1. Datapack создает scoreboard с названием "⏰ Игровое время"
2. Каждый тик (20 раз в секунду) обновляется время
3. Scoreboard отображается справа на экране для всех игроков
4. Показывается:
   - ⏰ Часы (0-23)
   - ⏰ Минуты (0-59)
   - ⏰ Время дня (0-24000, игровые тики)

## Формат времени

- **0-6000**: Утро (6:00 - 12:00)
- **6000-12000**: День (12:00 - 18:00)
- **12000-18000**: Вечер (18:00 - 0:00)
- **18000-24000**: Ночь (0:00 - 6:00)

## Ручная активация

Если datapack не загрузился автоматически, выполните в игре:
```
/reload
```

Или в консоли сервера:
```
reload
```

## Отключение

Чтобы временно скрыть scoreboard:
```
/scoreboard objectives setdisplay sidebar
```

Чтобы снова показать:
```
/scoreboard objectives setdisplay sidebar gametime_display
```

Чтобы полностью удалить:
```
/scoreboard objectives remove gametime_display
```
